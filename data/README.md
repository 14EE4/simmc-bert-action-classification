# Preprocessed CSV Data

이 폴더는 SIMMC 기반 BERT action classification 실험에 사용할 전처리 CSV 파일을 보관하는 위치입니다.

파일:

- `Origin_Train_Data_10k.csv`
- `Train_Data_10k_2.csv`
- `Middle_Test_Data.csv`
- `Final_Test_Data.csv`

## 라이선스

이 폴더에 포함되는 CSV 파일은 SIMMC 데이터셋을 기반으로 전처리한 데이터입니다. 비상업적 연구 및 교육 목적으로만 공유하며, CC BY-NC-SA 4.0 라이선스 조건을 따릅니다.

원본 SIMMC 저장소: <https://github.com/facebookresearch/simmc>
